# K-SUN

**Description:**
K-SUN is a fictitious business initiative aimed at promoting the creation of a solar vehicle brand. This project showcases a vision for a sustainable future through affordable, eco-friendly transportation. It offers information about the project, recruitment of experts, crowdfunding, and more.

**Features:**
- Informational page about solar vehicles.
- Crowdfunding system to support the initiative.
- Recruitment of experts in sustainable energy.

**Technologies Used:**
- HTML
- CSS
- JavaScript
- Node.js
- MySQL
- PHP
- Generative Image AI (Ideogram)

**Installation:**
1. Clone the repository:
    ```bash
    git clone https://github.com/KNBDevs/k-sun.git
    ```
2. Set up the environment and database:
    - Create a MySQL database.
    - Configure the `.env` file with the appropriate credentials.

3. Install project dependencies:
    ```bash
    npm install
    ```

4. Run the server:
    ```bash
    npm start
    ```
