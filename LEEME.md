# K-SUN

**Descripción:**
K-SUN es una iniciativa ficticia empresarial que busca promover la creación de una marca de vehículos solares. Este proyecto está orientado a mostrar una visión de futuro sostenible a través de medios de transporte asequibles y respetuosos con el medio ambiente. Ofrece información sobre el proyecto, reclutamiento de expertos, crowdfunding, entre otros.

**Características:**
- Página informativa sobre vehículos solares.
- Sistema de crowdfunding para apoyar la iniciativa.
- Reclutamiento de expertos en energías sostenibles.

**Tecnologías utilizadas:**
- HTML
- CSS
- JavaScript
- Node.js
- MySQL
- PHP
- IA Generativa de Imágenes (Ideogram)

**Instalación:**
1. Clonar el repositorio:
    ```bash
    git clone https://github.com/KNBDevs/k-sun.git
    ```
2. Configurar el entorno y base de datos:
    - Crear base de datos en MySQL.
    - Configurar los archivos `.env` con las credenciales apropiadas.

3. Instalar las dependencias del proyecto:
    ```bash
    npm install
    ```

4. Ejecutar el servidor:
    ```bash
    npm start
    ```
